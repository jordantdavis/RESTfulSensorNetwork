API Reference

Resources Overview

Resource Description
POST /device/register Creates a new GCM registered device with a set of available sensors.
POST /device/unregister Removes an existing GCM registered device.
POST /schedule/create Creates a new schedule containing sensors to sample for a given duration at a specified frequency.

Resource Details

POST /device/register

Creates a new GCM registered device with a set of available sensors.

Parameters

registrationId (string)
(required)
A Google Cloud Messaging provided registrationID for the new device.
availableSensors (array[string])
(required)
An array of sensors names available on the new device. The possible values for sensors are: accelerometer, gyroscope, humidity, light, location, magnetometer, pressure, proximity, and/or temperature.

POST /device/unregister

Removes an existing GCM registered device.

Parameters

registrationId (string)
(required)
A Google Cloud Messaging provided registrationID for an existing device.

POST /schedule/create

Creates a new schedule containing sensors to sample for a given duration at a specified frequency.

Parameters

schedules (array[Schedule])
(required)
An array of schedule objects.

Types

Schedule
sensorName (string)
(required)
The name of a sensor to sample. The possible values for sensors are: accelerometer, gyroscope, humidity, light, location, magnetometer, pressure, proximity, and/or temperature.
startTime (int)
(required)
Sampling start time in epoch time (seconds).
endTime (int)
(required)
Sampling end time in epoch time (seconds).
frequency (int/float)
(required)
Sampling speed (in Hz).
