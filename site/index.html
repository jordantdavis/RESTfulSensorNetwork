RESTful Sensor Network

The crowdsourced Android smartphone sensor data aggregator.

About

RSN is a wireless sensor network architecture which aggregates data from Android devices based on distributed schedules. Users of Android devices can register themselves with RSN by publishing a list of sensors the user is willing to provide samples from. In return, a user can publish sensor sampling schedules to all other connected devices and later retrieve that data from RSN.
